
A C program contains the following declaration
int x[8]= {10,20,30,40,50,60,70,80};

● What is the meaning of x?
    ->'x' is a integer type array decleration.

● What is the meaning of (x+2)?
    ->this gives the value at 3rd element in the array i.e; 30

● What is the value of *x?
    ->this gives the value of 1st element in the array i.e; 10

● What is the value of (*x+2)?
    ->this gives the value of *x is 10 and 2 is added to that values gives the ans as 12.

● What is the value of *(x+2)?
    ->this gives the element at 3rd position in the array i.e; 30
